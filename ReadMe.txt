How to install our project:

	1: download a personal web server application.
		As developers, our team used XAMPP and easyphp
	2: next download and install github.
	3: clone of our project, found at https://github.com/dfranc3373/CSProject.
	4a: copy the CSProject folder into the root of the XAMPP server
		OR
	4b: set the clone directly in the root of the XAMPP server
	5: start the XAMPP server
	6: proceed to http://localhost/csproject/www/index.php
	7: sign into the online application.
	We used test//test for most of the work.
